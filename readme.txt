
GraphicsEngine
==============

Requirements:

  Platform            Win32
  Language            C++
  GraphicsAPI         OpenGL v2.1 or above
  External Libraries  glLoadGen, GLM, LodePNG
  Features etc.       Window,
                      AssetManager,
                      GraphicsContext,
                      Vertex- and IndexBuffer,
                      Texture (.png),
                      Shader, 
                      Sprite,
                      SpriteBatch,
                      Textured 3D Models (.obj)
