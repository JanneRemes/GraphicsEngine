
GraphicsEngine
==============

Requirements:

  Platform            Win32
  Language            C++
  GraphicsAPI         OpenGL v2.1 or above
  External Libraries  GLEW, GLM, LodePNG
  Features etc.       Window Class,
                      Filemanager,
                      GraphicsContext,
                      Vertex- and Index Buffers,
                      Textures,
                      Shaders,
                      Sprite Class,
                      Sprite Batching,
                      User Input System
